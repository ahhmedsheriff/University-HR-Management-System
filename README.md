# University HR Management System

A full-stack University Human Resources Management System built with SQL Server and ASP.NET, covering employee management, payroll, attendance, leave workflows, and performance evaluations.

## Overview
This project automates core HR workflows for a university setting, combining a structured SQL Server database with stored procedures and views, alongside a C#/ASP.NET web interface for employees and administrators.

## Features
- Employee and admin management
- Automated payroll processing
- Attendance tracking
- Leave request and approval workflows
- Performance evaluations
- Real-time employee and admin interactions through a web interface
- Database architecture with stored procedures and views

## Tech Stack
- SQL Server (Database design, stored procedures, views)
- C#
- ASP.NET
- CSS

## How to Run
1. Clone the repository
2. Open `M3.sln` in Visual Studio
3. Set up the SQL Server database using `final_implementation.sql`
4. Configure the database connection string
5. Run the project
